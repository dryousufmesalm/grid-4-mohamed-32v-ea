## Requirements

MetaTrader 5. YM library in MQL5/Include/YM.

## Installation

Copy Grid_4_Mohamed_32v.mq5 into MQL5/Experts/.

## Compilation

Compile in MetaEditor (F7). Include path must include terminal MQL5 folder.

## Usage

Attach to a chart. Configure grid inputs (lot, step, etc.).
